# Statman

## About
Discord Status manager  ||  CLI 

## Requirements

  - One verified Discord Account { Token }
  - Python3 
  - Git
 
## Installation
  
  Note : Use pip3 for Linux :)
  
   ```
   pip install rich requests
   ```
   
   ```
   git clone https://github.com/mastercodermerwin/Statman.git
   ```
   
## Running
  
  Note : Mention python3 if in Linux :)
  
  ```
  cd Statman
  ```
  
  ```
  python main.py
  ```
